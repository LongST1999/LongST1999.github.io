    # 7. Window.opener

    页面使用target="_blank"时,将会允许新页面使用window.opener(它会返回对创建该窗口的 Window 对象的引用)。这会对页面的安全和性能造成影响，可以使用 rel="noopener" 或者 rel="noreferrer" 来避免。

    # 9. 图标缓存破坏(Favicon cache busting)

    你可以强制刷新你网站的图标，通过在文件名中添加?v=2来强制浏览器下载新版本。

    这在生产中特别有用，可以确保用户获得新版本。

    <link rel="icon" href="/favicon.ico?v=2" />

    # 13. download属性

    您可以给<a>链接添加download属性来下载文件而不是导航到它(instead of navigating to it)

    <a href='path/to/file' download>
  Download
</a>

    # 14. 性能提示(Performance tip)

    使用 .webp 图像格式缩小图像大小并提高网站的性能。

    <picture>
  <!-- load .webp image if supported -->
  <source srcset="logo.webp" type="image/webp">
  
  <!-- 
	Fallback if `.webp` images or <picture> tag 
	not supported by the browser.
  -->
  <img src="logo.png" alt="logo">
</picture>
